Rate format → fixed viewport → candle period
/day
14 candles · 2-week viewport
Each candle = 1 day's actual margin vs projected $/day. Stack shows each transaction on that day with its $/day contribution and drift.
Scrub range: data start → future projection
/mo ← active
12 candles · 1-year viewport
Each candle = 1 month's actual margin vs projected $/mo. Stack shows each commitment at its monthly rate with $/mo drift. Hits and Boosts labeled.
Scrub range: data start → future projection
/yr
10 candles · 1-decade viewport
Each candle = 1 year's actual margin vs projected $/yr. Stack shows annual totals per category. Long-horizon planning view — see how the projection evolves over years.
Scrub range: data start → future projection
Drift column convention: +$X/mo = spending that much more than projected (behind). −$X/mo = spending that much less than projected (ahead). Sign reflects impact on margin — positive drift on a commitment means it cost more than expected.
